# belajar-react-use-reducer
Contoh penggunaan hook `useReducer` di ReactJS.

# Penggunaan
### Clone
```bash
git clone https://github.com/hadihammurabi/belajar-react-use-reducer.git
```

### Instalasi
```bash
npm install
```

atau 

```bash
yarn install
```

atau

```bash
pnpm install
```

### Menjalankan
```bash
npm run dev
```

atau 

```bash
yarn dev
```

atau

```bash
pnpm dev
```
